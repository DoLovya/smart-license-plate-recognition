# 开发环境说明

## Python 环境

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 算法环境

```bash
cd algorithms
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Qt 客户端

- 安装 Qt 6.5+ 与 CMake 3.20+
- 进入 `frontend/qt_client/` 目录执行 CMake 构建
